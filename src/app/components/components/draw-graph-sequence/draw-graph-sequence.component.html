Princípio de Havel-Hakimi

O algoritmo de Havel-Hakimi é utilizado para determinar se uma sequência de graus pode formar um grafo simples. A sequência de graus representa a quantidade de arestas que cada vértice de um grafo deve ter. O algoritmo opera da seguinte forma:

  1. Ordene a sequência de graus de forma decrescente.
  2. Retire o primeiro elemento X da sequência (o maior valor).
  3. Subtraia 1 dos próximos X elementos da sequência (ou seja, conecte o vértice com os próximos vértices de maior grau).
  4. Repita o processo até que todos os elementos sejam 0 ou se identifique uma contradição (quando não houver vértices suficientes para conectar).

Se a sequência se reduzir a todos zeros, a sequência de graus é válida e um grafo pode ser construído a partir dela.

Exemplo: Sequência de Grau [4, 3, 3, 3, 1]

A sequência de grau [4, 3, 3, 3, 1] descreve um grafo com 5 vértices e as seguintes conexões:

V1 (grau 4)
V2 (grau 3)
V3 (grau 3)
V4 (grau 3)
V5 (grau 1)
V1 se conecta a V2, V3, V4, V5
V2 se conecta a V3, V4
V3 se conecta a V4

O grafo resultante é um grafo simples e não dirigido que satisfaz a sequência de graus dada.
